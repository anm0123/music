# music
基于微信小程序 -- 音乐播放小程序<br>
由于音乐存储在自己的云空间，可能导致音乐的连接不可用。只需更换音乐连接即可（将自己的本地音乐上传到自己的微信小程序云空间，在更换链接）。<br>
